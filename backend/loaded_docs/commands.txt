# Minecraft Java Edition 指令使用手冊 (RAG 知識庫)

## 1. 基礎概念與變數定義
在學習指令前，必須理解以下通用參數的定義：

* **<targets> (目標選擇器):** 指定指令的執行對象。
    * `@a`: 所有玩家 (All players)
    * `@p`: 最近的玩家 (Nearest player)
    * `@r`: 隨機玩家 (Random player)
    * `@e`: 所有實體 (All entities，包含生物、掉落物、盔甲架等)
    * `@s`: 指令執行者自身 (Self)
    * *進階用法:* `@e[type=zombie, limit=1]` (只選擇一隻殭屍)、`@a[distance=..10]` (10格內的玩家)。
* **<pos> / <location> (座標):** 指定空間位置。
    * `x y z`: 絕對座標 (例如 `100 64 -200`)。
    * `~ ~ ~`: 相對座標 (相對於執行者的位置，`~1` 代表向正方向偏移 1 格)。
    * `^ ^ ^`: 局部座標 (相對於執行者的「視線」方向，`^ ^ ^5` 代表面前 5 格)。
* **<item> (物品ID):** 物品的識別名稱，通常格式為 `namespace:item_name` (例如 `minecraft:diamond_sword`)。
* **<nbt> (NBT標籤):** 實體的詳細數據，使用 JSON 格式 (例如 `{Unbreakable:1b}` 表示物品不可破壞)。

## 2. 指令詳細列表

### /advancement
* **描述:** 加入、移除或查詢玩家的進度 (成就)。
* **語法:** `/advancement <grant|revoke> <targets> <only|from|through|until|everything> [advancement]`
* **參數:**
    * `grant/revoke`: 給予或移除。
    * `only`: 僅操作指定的單一進度。
    * `everything`: 操作所有進度。
* **範例:** `/advancement grant @a only minecraft:story/iron_tools` (給予所有玩家「鐵器時代」進度)。

### /attribute
* **描述:** 修改或查詢生物的屬性 (如最大血量、攻擊力)。
* **語法:** `/attribute <targets> <attribute> <get|base|modifier> ...`
* **範例:** `/attribute @s minecraft:generic.max_health base set 40` (將玩家基礎血量設為 40，即 20 顆心)。

### /ban
* **描述:** 將玩家加入封鎖清單 (黑名單)。
* **語法:** `/ban <targets> [reason]`
* **範例:** `/ban BadPlayer 作弊行為` (封鎖 BadPlayer 並顯示理由)。

### /ban-ip
* **描述:** 將 IP 位址加入封鎖清單。
* **語法:** `/ban-ip <targets|ip> [reason]`
* **範例:** `/ban-ip 192.168.1.1`。

### /banlist
* **描述:** 顯示封鎖清單。
* **語法:** `/banlist [ips|players]`
* **範例:** `/banlist players` (列出被封鎖的玩家)。

### /bossbar
* **描述:** 建立、修改或查詢 Boss 血條 (畫面頂端的狀態條)。
* **語法:** `/bossbar <add|get|list|remove|set> ...`
* **範例:**
    1. `/bossbar add my_bar "Raid Boss"` (新增 ID 為 my_bar 的血條)。
    2. `/bossbar set my_bar value 100` (設定數值)。
    3. `/bossbar set my_bar players @a` (讓所有玩家看到)。

### /clear
* **描述:** 從玩家物品欄中刪除物品。
* **語法:** `/clear [targets] [item] [maxCount]`
* **範例:** `/clear @a minecraft:tnt` (清除所有人身上的 TNT)。

### /clone
* **描述:** 將特定區域的方塊複製到另一處。
* **語法:** `/clone <begin> <end> <destination> [maskMode] [cloneMode]`
* **參數:**
    * `<begin> <end>`: 起始與結束座標 (定義一個長方體)。
    * `<destination>`: 貼上的目標位置 (左下角)。
* **範例:** `/clone 0 60 0 10 70 10 100 60 100 replace normal`。

### /damage
* **描述:** 對指定實體造成傷害。
* **語法:** `/damage <targets> <amount> [damageType]`
* **範例:** `/damage @e[type=creeper] 10 minecraft:lava` (對苦力怕造成 10 點熔岩傷害)。

### /data
* **描述:** 取得、合併、修改或移除實體或方塊的 NBT 資料 (高階指令)。
* **語法:** `/data <get|merge|modify|remove> <block|entity|storage> <target> [path] ...`
* **範例:** `/data merge entity @e[type=pig,limit=1] {Saddle:1b}` (給最近的豬裝上鞍)。

### /datapack
* **描述:** 控制或列出載入的資料包。
* **語法:** `/datapack <disable|enable|list> <name> ...`
* **範例:** `/datapack list`。

### /debug
* **描述:** 開始或結束除錯分析 (用於效能測試)。
* **語法:** `/debug <start|stop|function>`
* **範例:** `/debug start`。

### /defaultgamemode
* **描述:** 更改伺服器預設的遊戲模式 (對新加入玩家生效)。
* **語法:** `/defaultgamemode <mode>`
* **範例:** `/defaultgamemode survival`。

### /deop
* **描述:** 撤銷玩家的管理員 (OP) 權限。
* **語法:** `/deop <targets>`
* **範例:** `/deop PlayerName`。

### /dialog
* **描述:** 向玩家展示或清除 NPC 對話框 (主要用於基岩版或特定教育版功能，Java版少用)。
* **語法:** `/dialog <open|change> ...`

### /difficulty
* **描述:** 設定遊戲難易度。
* **語法:** `/difficulty <peaceful|easy|normal|hard>`
* **範例:** `/difficulty hard` (設定為困難)。

### /effect
* **描述:** 給予或移除生物的狀態效果 (如加速、中毒)。
* **語法:** `/effect <give|clear> <targets> [effect] [seconds] [amplifier] [hideParticles]`
* **範例:** `/effect give @s minecraft:speed 30 1 true` (給予自己 30 秒的速度 II 效果，且不顯示粒子)。
    * *註:* amplifier 0 代表等級 1，amplifier 1 代表等級 2。

### /enchant
* **描述:** 為玩家手持的物品附魔。
* **語法:** `/enchant <targets> <enchantment> [level]`
* **範例:** `/enchant @s minecraft:sharpness 5` (給手持武器附魔鋒利 V)。

### /execute
* **描述:** 極為強大的指令，可改變執行者、位置或條件來執行後續指令。
* **語法:** `/execute <subcommands> run <command>`
* **常用子指令:**
    * `as <targets>`: 以某人的身分。
    * `at <targets>`: 在某人的位置。
    * `if <condition>`: 如果條件成立。
* **範例:** `/execute as @a at @s run setblock ~ ~-1 ~ minecraft:gold_block` (讓所有玩家在自己腳下放置金磚)。

### /experience (別名 /xp)
* **描述:** 給予、移除或查詢玩家經驗。
* **語法:** `/experience <add|set|query> <targets> <amount> [points|levels]`
* **範例:** `/experience add @a 5 levels` (給所有人 5 等級)。

### /fill
* **描述:** 用特定方塊填滿一個區域。
* **語法:** `/fill <from> <to> <block> [destroy|hollow|keep|outline|replace]`
* **範例:** `/fill ~-5 ~-5 ~-5 ~5 ~5 ~5 minecraft:air` (清除身邊 5x5x5 範圍的方塊，填充空氣)。

### /fillbiome
* **描述:** 將區域設定為指定生態域 (改變草色、天氣行為等)。
* **語法:** `/fillbiome <from> <to> <biome>`
* **範例:** `/fillbiome ~-10 ~-10 ~-10 ~10 ~10 ~10 minecraft:desert` (將附近改為沙漠生態)。

### /forceload
* **描述:** 強制載入區塊 (即使沒人也不會被卸載)。
* **語法:** `/forceload <add|remove|query> <pos>`
* **範例:** `/forceload add ~ ~` (強制載入當前區塊)。

### /function
* **描述:** 執行資料包中的函數 (.mcfunction 檔案)。
* **語法:** `/function <name>`
* **範例:** `/function my_datapack:start_game`。

### /gamemode
* **描述:** 更改玩家的遊戲模式。
* **語法:** `/gamemode <survival|creative|adventure|spectator> [targets]`
* **範例:** `/gamemode creative @s` (讓自己變成創造模式)。

### /gamerule
* **描述:** 更改或查詢遊戲規則 (全域設定)。
* **語法:** `/gamerule <rule> [value]`
* **常用規則:**
    * `keepInventory`: 死亡不掉落。
    * `doDaylightCycle`: 時間是否流動。
    * `mobGriefing`: 生物是否破壞方塊。
* **範例:** `/gamerule keepInventory true`。

### /give
* **描述:** 給予玩家物品。
* **語法:** `/give <targets> <item> [amount]`
* **範例:** `/give @s minecraft:diamond_sword 1`。

### /help
* **描述:** 查詢指令語法。
* **語法:** `/help [command]`
* **範例:** `/help give`。

### /item
* **描述:** 修改容器 (箱子) 或實體身上的物品。
* **語法:** `/item <modify|replace> <block|entity> ...`
* **範例:** `/item replace entity @s armor.head with minecraft:diamond_helmet` (強制給自己戴上鑽石頭盔)。

### /kick
* **描述:** 將玩家踢出伺服器。
* **語法:** `/kick <targets> [reason]`
* **範例:** `/kick Steve 伺服器維護中`。

### /kill
* **描述:** 清除/殺死實體。
* **語法:** `/kill <targets>`
* **範例:** `/kill @e[type=!player]` (殺死除玩家以外的所有生物)。

### /list
* **描述:** 列出目前在線的玩家。
* **語法:** `/list [uuids]`。

### /locate
* **描述:** 尋找最近的結構或生態域座標。
* **語法:** `/locate <structure|biome|poi> <id>`
* **範例:** `/locate structure minecraft:village_plains` (尋找平原村莊)。

### /loot
* **描述:** 模擬戰利品生成並放入容器或給予玩家。
* **語法:** `/loot <target> <source>`
* **範例:** `/loot give @s fish` (模擬釣魚掉落物給玩家)。

### /me
* **描述:** 在聊天欄顯示動作訊息。
* **語法:** `/me <message>`
* **範例:** `/me 正在思考` (輸出: * Player 正在思考)。

### /msg (別名 /tell, /w)
* **描述:** 傳送私訊給玩家。
* **語法:** `/msg <targets> <message>`
* **範例:** `/msg Alex 秘密基地在哪？`。

### /op
* **描述:** 授予玩家管理員權限。
* **語法:** `/op <targets>`。

### /pardon / /pardon-ip
* **描述:** 解除封鎖 (玩家或 IP)。
* **語法:** `/pardon <targets>`。

### /particle
* **描述:** 在指定位置播放粒子效果。
* **語法:** `/particle <name> <pos> [delta] [speed] [count]`
* **範例:** `/particle minecraft:flame ~ ~ ~ 0.5 0.5 0.5 0.1 100` (在身邊產生 100 個火焰粒子)。

### /place
* **描述:** 直接在世界中放置結構、地物或拼圖。
* **語法:** `/place <feature|structure|template> ...`
* **範例:** `/place structure minecraft:village_plains ~ ~ ~`。

### /playsound
* **描述:** 播放音效。
* **語法:** `/playsound <sound> <source> <targets> <pos> [volume] [pitch]`
* **範例:** `/playsound minecraft:entity.experience_orb.pickup master @a ~ ~ ~ 1 1`。

### /publish
* **描述:** 將單人世界開放為區域網路 (LAN) 伺服器。
* **語法:** `/publish [port]`。

### /random
* **描述:** 生成隨機數值 (主要用於指令邏輯)。
* **語法:** `/random value <range>` 或 `/random roll <range>`。

### /recipe
* **描述:** 給予或剝奪玩家的合成配方。
* **語法:** `/recipe <give|take> <targets> <recipe>`
* **範例:** `/recipe give @a *` (解鎖所有配方)。

### /reload
* **描述:** 重新載入資料包 (Datapacks) 和標籤。
* **語法:** `/reload` (修改資料包後必須執行此指令才生效)。

### /ride
* **描述:** 強制實體騎乘另一個實體。
* **語法:** `/ride <target> mount <vehicle>`
* **範例:** `/ride @s mount @e[type=pig,limit=1]` (騎上最近的豬)。

### /rotate
* **描述:** 更改實體面向的角度。
* **語法:** `/rotate <target> <facing|rotation> ...`

### /save-all / /save-off / /save-on
* **描述:** 伺服器存檔控制。
    * `/save-all`: 立即寫入硬碟。
    * `/save-off`: 停止自動存檔。
    * `/save-on`: 開啟自動存檔。

### /say
* **描述:** 廣播訊息 (所有人都看得到)。
* **語法:** `/say <message>`。

### /schedule
* **描述:** 安排函數在未來的某個時間點執行。
* **語法:** `/schedule function <function> <time>`
* **範例:** `/schedule function my_pack:event 10s` (10秒後執行函數)。

### /scoreboard
* **描述:** 管理計分板 (用於小遊戲製作、統計數據)。
* **語法:** `/scoreboard <objectives|players> ...`
* **常用操作:**
    1. 建立記分項: `/scoreboard objectives add Coins dummy "金幣"`
    2. 顯示在右側: `/scoreboard objectives setdisplay sidebar Coins`
    3. 加分: `/scoreboard players add @s Coins 10`

### /seed
* **描述:** 顯示世界種子碼 (Seed)。
* **語法:** `/seed`。

### /setblock
* **描述:** 放置單一方塊。
* **語法:** `/setblock <pos> <block> [destroy|keep|replace]`
* **範例:** `/setblock ~ ~-1 ~ minecraft:diamond_block` (將腳下變為鑽石磚)。

### /setidletimeout
* **描述:** 設定掛機踢出時間 (分鐘)。
* **語法:** `/setidletimeout <minutes>`。

### /setworldspawn
* **描述:** 設定世界重生點 (所有新玩家的出生地)。
* **語法:** `/setworldspawn [pos]`。

### /spawnpoint
* **描述:** 設定特定玩家的重生點 (睡床位置)。
* **語法:** `/spawnpoint [targets] [pos]`
* **範例:** `/spawnpoint @s ~ ~ ~`。

### /spectate
* **描述:** 讓處於旁觀模式的玩家進入某個實體的視角。
* **語法:** `/spectate [target] [player]`。

### /spreadplayers
* **描述:** 將實體隨機散布在某個區域 (常用於大逃殺遊戲)。
* **語法:** `/spreadplayers <center> <spreadDistance> <maxRange> <respectTeams> <targets>`。

### /stop
* **描述:** 安全關閉伺服器。
* **語法:** `/stop`。

### /stopsound
* **描述:** 停止正在播放的聲音。
* **語法:** `/stopsound <targets> [source] [sound]`。

### /summon
* **描述:** 生成實體。
* **語法:** `/summon <entity> [pos] [nbt]`
* **範例:** `/summon minecraft:zombie ~ ~ ~ {HandItems:[{id:"minecraft:diamond_sword",Count:1b},{}], ArmorItems:[{},{},{},{id:"minecraft:diamond_helmet",Count:1b}]}` (生成一隻全副武裝的殭屍)。

### /tag
* **描述:** 給實體加上臨時標籤 (Tag)，方便選擇器篩選。
* **語法:** `/tag <targets> <add|remove|list> <name>`
* **範例:**
    1. `/tag @s add Runner` (給自己加 Runner 標籤)。
    2. `/execute as @a[tag=Runner] ...` (只對 Runner 執行)。

### /team
* **描述:** 隊伍系統管理 (同隊不會互相傷害、顏色顯示等)。
* **語法:** `/team <add|join|leave|modify|remove> ...`
* **範例:**
    1. `/team add RedTeam "紅隊"`
    2. `/team join RedTeam @s`
    3. `/team modify RedTeam color red`

### /teammsg (別名 /tm)
* **描述:** 僅傳送訊息給同隊伍的玩家。
* **語法:** `/teammsg <message>`。

### /teleport (別名 /tp)
* **描述:** 傳送實體。
* **語法:**
    1. `/tp <destination>` (傳送到某人那)
    2. `/tp <location>` (傳送到座標)
    3. `/tp <targets> <destination>` (把 A 傳送到 B)
* **範例:** `/tp @a @s` (把所有人傳送到我這裡)。

### /tellraw
* **描述:** 傳送 JSON 格式的複雜訊息 (可點擊、懸停顯示)。
* **語法:** `/tellraw <targets> <json>`
* **範例:** `/tellraw @a {"text":"點我領取鑽石","color":"gold","clickEvent":{"action":"run_command","value":"/give @s diamond 1"}}`。

### /tick
* **描述:** 控制遊戲刻 (Tick) 速率 (可暫停時間、加速或慢動作)。
* **語法:** `/tick <rate|freeze|step|unfreeze> ...`
* **範例:** `/tick rate 200` (加速 10 倍)。

### /time
* **描述:** 更改或查詢世界時間。
* **語法:** `/time <add|query|set> <value>`
* **參數:**
    * `day`: 1000
    * `noon`: 6000
    * `night`: 13000
    * `midnight`: 18000
* **範例:** `/time set day`。

### /title
* **描述:** 在螢幕中央顯示大標題。
* **語法:** `/title <targets> <title|subtitle|actionbar|times|clear|reset> <json>`
* **範例:**
    1. `/title @a times 10 70 20` (設定淡入淡出時間)。
    2. `/title @a title {"text":"遊戲開始!", "color":"red"}`。

### /trigger
* **描述:** 讓普通玩家觸發計分板修改 (通常用於非 OP 玩家執行特定功能)。
* **語法:** `/trigger <objective> [add|set]`。

### /weather
* **描述:** 設定天氣。
* **語法:** `/weather <clear|rain|thunder> [duration]`
* **範例:** `/weather clear` (晴天)。

### /whitelist
* **描述:** 管理白名單 (只允許名單內玩家進入)。
* **語法:** `/whitelist <add|remove|on|off|list>`。

### /worldborder
* **描述:** 設定世界邊界 (玩家無法超出的範圍)。
* **語法:** `/worldborder <center|set|add|get> ...`
* **範例:** `/worldborder set 100 20` (將邊界縮小為 100格，並在 20秒內完成)。
